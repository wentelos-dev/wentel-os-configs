# 🚀 WentelOS (Official Repository)

WentelOS, Ubuntu Server (24.04 LTS) bazasında hazırlanmış, XFCE masaüstü mühitini istifadə edən və vizual olaraq Windows 10-a bənzədilmiş özəl bir Linux paylanmasıdır.

## 🎯 Layihənin Məqsədi
- Köhnə kompüterlər üçün yüksək performans.
- Windows istifadəçiləri üçün tanış interfeys.
- Minimum resurs istifadəsi ilə maksimum sürət.

## 📂 Repository Strukturu
- `/config`: XFCE panel və sistem tənzimləmələri.
- `/themes`: Windows 10 vizual mövzuları.
- `/scripts`: Avtomatik quraşdırma skriptləri.
